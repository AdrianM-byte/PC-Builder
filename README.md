## Overview

A Python-based application designed to automate PC configuration by selecting compatible components based on user-defined requirements.

The goal is to simulate a decision-making system that reduces manual research and simplifies the PC building process.

---

## Problem

Building a PC requires:
- Comparing multiple components  
- Ensuring compatibility  
- Evaluating performance vs budget  

This process is time-consuming and often inconsistent.

---

## Solution

This project aims to automate the process by:
- Applying decision-based logic for component selection  
- Filtering options based on constraints (budget, performance, compatibility)  
- Supporting multi-source comparison across vendors  

---
